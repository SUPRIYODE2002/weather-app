Weather App

A responsive and user-friendly Weather Application built using HTML, CSS, and JavaScript.
It uses a real-time weather API to display the current temperature, humidity, wind speed, and weather conditions of any city searched by the user.


Live Demo:-

Features:-

1.Search weather by city name

2. Live temperature, humidity, wind speed

3. Dynamic weather icons

4. Fully responsive UI

 5.Powered by Fetch API & async JavaScript

6.Error handling for invalid city names

7. Clean and modern interface



Tech Stack:-

HTML5 – Structure

CSS3 – Styling & layout

JavaScript (ES6) – Logic, API calls, DOM updates

OpenWeatherMap API – Real-time weather data
